# Contributing

Thanks for your interest in improving this template.

## How to Contribute

1. Fork this repository
2. Create a feature branch (`git checkout -b 20260504-your-feature`)
3. Make your changes
4. Run lint and tests: `ruff check . && python -m pytest tests/ -v`
5. Commit with conventional format: `feat:`, `fix:`, `docs:`, `chore:`
6. Open a pull request with a clear description of the change

## Reporting Issues

Open a GitHub Issue with:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Your OS and VS Code version

## Scope

This template focuses on VA development workflows with VS Code and GitHub Copilot. Changes should align with that purpose — agents, instructions, prompts, setup tooling, and compliance guidance.
